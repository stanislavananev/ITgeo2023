В качестве тренировочного задания реализовано приложение для отображения курсов валют. В данный момент работа приложения осуществляется следующим образом:
1) При помощи отдельного скрипта собираются данные курсов валют, находящиеся в открытом доступе. Частота обновления данных - 10 мин.
2) При загрузке нового набора данных (множество точек с названием валюты, кодом, значением, временем обновления), для каждой точки создается свой объект. 
3) в качестве private key выбран код валюты (сокращенное название типа USD), по которому все объекты, соответствующие одной валюте, группируются. 
4) Визуализация реализована при помощи библиотеки Apex Charts. Все точки данных соответствующие одной валюте отображаются в формате Значение/Время.
